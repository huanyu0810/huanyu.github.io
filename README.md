# 今天吃什么网站静态部署

这是 GitHub Pages 标准静态部署版本。

上传到 GitHub 时，请上传本文件夹里面的所有内容，而不是上传本文件夹本身。

必须包含：
- index.html
- 404.html
- .nojekyll
- assets 文件夹
- photos 文件夹
- food-hero.svg

部署地址说明：
- 如果仓库名是 `huanyu0810.github.io`，访问 `https://huanyu0810.github.io/`
- 如果仓库名是 `huanyu.github.io`，访问 `https://huanyu0810.github.io/huanyu.github.io/`

不要用 `https://github.com/...` 当成网站地址，那是 GitHub 仓库页面。
